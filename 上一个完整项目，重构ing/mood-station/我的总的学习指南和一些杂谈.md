# 干不同工作的语言选择
| 核心领域 | 具体应用场景（User Story） | 行业标准 / 首选语言 | 权威框架 / 技术栈示例 | 选型理由（Why） |
|---------|---------------------------|--------------------|----------------------|----------------|
| 后端与云原生（联网开发） | 构建高并发、微服务架构、企业级后台系统 | Java / Go | Java：Spring Boot / Spring Cloud<br>Go：Gin, gRPC, Kubernetes | Java：生态最完善，招聘最容易，适合复杂业务逻辑。<br>Go：云原生时代的 C 语言，启动快，并发性能极强，适合中间件和微服务。 |
| 后端与云原生（联网开发） | 初创项目、快速原型开发、全栈开发 | Python / Node.js（TypeScript） | Python：FastAPI, Django<br>Node：NestJS, Express | 开发速度极快。Node.js 允许前后端使用同一种语言（JavaScript / TypeScript）。 |
| 人工智能与算法（数据 / 科学） | AI 模型训练、数据分析、科学计算 | Python | PyTorch, TensorFlow, NumPy, Pandas | 拥有垄断级的生态库。在 AI 领域，Python 是唯一的“通用语”。 |
| 人工智能与算法（数据 / 科学） | 算法竞赛（OI / ACM）、底层高性能算子实现 | C++ | STL, CUDA（NVIDIA） | 追求极致的运行时效率和内存控制。大多数 Python 的 AI 库底层都是 C++ 编写。 |
| 系统与嵌入式（底层开发） | 操作系统内核、驱动程序、IoT 设备、单片机 | C / C++ | Linux Kernel, RTOS | 可直接操作硬件，资源占用极低，是现代计算机世界的基石。 |
| 系统与嵌入式（底层开发） | 高性能命令行工具、系统级基础设施 | Rust | Cargo, Tokio | 内存安全（无 GC），性能媲美 C++ 但更安全，是系统编程的未来趋势。 |
| 移动端开发（App） | 追求完美体验的原生 iOS / Android 应用 | Swift（iOS） / Kotlin（Android） | SwiftUI, Jetpack Compose | 官方御用语言，可调用最新系统 API，性能最佳。 |
| 移动端开发（App） | 跨平台应用（一套代码跑双端） | Dart / JavaScript | Dart：Flutter<br>JS：React Native | 在开发效率和性能之间取得良好平衡，适合大多数商业 App。 |
| 游戏开发（互动娱乐） | 3A 大作、高画质 PC / 主机游戏 | C++ | Unreal Engine（虚幻引擎） | 游戏对性能极其敏感，C++ 是该领域的绝对霸主。 |
| 游戏开发（互动娱乐） | 独立游戏、手游、快速开发 | C# | Unity 3D | 门槛低于 C++，Unity 引擎市场占有率极高。 |
| 前端开发（网页 / 界面） | 浏览器网页、Web 应用、小程序 | JavaScript / TypeScript | React, Vue, Angular | 浏览器只认识 JS；TypeScript 是 JS 的超集，适合大型项目，是当前前端标准。 |

# HTML & CSS 学习

## 📖 学习内容

### HTML5 基础
- HTML 文档结构
- 常用标签（div, span, p, h1-h6, a, img, ul, ol, table 等）
- 语义化标签（header, nav, main, article, section, aside, footer）
- 表单元素（input, textarea, select, button）
- HTML5 新特性（video, audio, canvas, svg）

### CSS3 基础
- CSS 选择器（标签、类、ID、属性、伪类、伪元素）
- 盒模型（margin, padding, border）
- 布局方式（Flexbox, Grid）
- 定位（static, relative, absolute, fixed, sticky）
- 响应式设计（媒体查询）
- CSS3 新特性（动画、过渡、变换、渐变）

## 🎯 学习目标

- [ ] 掌握 HTML5 基本结构和常用标签
- [ ] 理解 CSS 盒模型和布局原理
- [ ] 能够使用 Flexbox 和 Grid 进行页面布局
- [ ] 实现响应式网页设计
- [ ] 掌握 CSS 动画和过渡效果

## 💻 实践项目

1. **个人简历页面** - 使用 HTML 和 CSS 创建一个美观的个人简历
2. **产品展示页** - 使用 Flexbox 布局创建产品卡片展示页
3. **响应式导航栏** - 实现适配移动端和桌面端的导航菜单
4. **博客文章页面** - 使用语义化标签构建博客文章页面

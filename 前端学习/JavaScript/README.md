# JavaScript 学习

## 📖 学习内容

### JavaScript 基础
- 变量与数据类型（String, Number, Boolean, Object, Array, null, undefined）
- 运算符与表达式
- 条件语句（if/else, switch）
- 循环语句（for, while, do-while）
- 函数（函数声明、函数表达式、箭头函数）
- 作用域与闭包
- this 关键字

### JavaScript 进阶
- ES6+ 新特性（let/const, 解构赋值, 模板字符串, 默认参数）
- 数组方法（map, filter, reduce, forEach）
- Promise 与异步编程
- async/await
- 模块化（import/export）
- 类与面向对象编程

### DOM 操作
- 获取和修改 DOM 元素
- 事件监听与处理
- 表单验证
- 动态创建和删除元素

### Web APIs
- Fetch API（网络请求）
- Local Storage / Session Storage
- 浏览器 API（Geolocation, Notification 等）

## 🎯 学习目标

- [ ] 掌握 JavaScript 基本语法和数据类型
- [ ] 理解函数、作用域和闭包
- [ ] 熟练使用 ES6+ 新特性
- [ ] 掌握 DOM 操作和事件处理
- [ ] 理解异步编程和 Promise
- [ ] 能够使用 Fetch API 进行网络请求

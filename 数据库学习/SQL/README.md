# SQL 数据库学习

## 📖 学习内容

### SQL 基础
- 数据库基本概念
- SQL 语法基础
- 数据类型
- 创建和管理数据库
- 创建和管理表

### 数据操作
- SELECT 查询（WHERE, ORDER BY, LIMIT）
- 聚合函数（COUNT, SUM, AVG, MAX, MIN）
- 分组查询（GROUP BY, HAVING）
- 连接查询（INNER JOIN, LEFT JOIN, RIGHT JOIN）
- 子查询
- INSERT 插入数据
- UPDATE 更新数据
- DELETE 删除数据

### 数据库设计
- 实体关系图（ER 图）
- 数据库范式（1NF, 2NF, 3NF）
- 主键和外键
- 索引优化
- 数据库约束

### MySQL
- MySQL 安装和配置
- 用户和权限管理
- 存储引擎（InnoDB, MyISAM）
- 事务处理（ACID）
- 存储过程和触发器
- 视图

### PostgreSQL
- PostgreSQL 特性
- JSON 数据类型
- 全文搜索
- 窗口函数

### 性能优化
- 查询优化
- 索引策略
- 执行计划分析
- 数据库备份和恢复

## 🎯 学习目标

- [ ] 掌握 SQL 基本语法和查询
- [ ] 能够设计规范的数据库结构
- [ ] 理解数据库范式和关系设计
- [ ] 掌握索引和查询优化技巧
- [ ] 理解事务和 ACID 特性
- [ ] 能够进行数据库备份和恢复

## 💻 实践项目

1. **学生成绩管理系统** - 设计学生、课程、成绩表及关系
2. **电商数据库** - 设计用户、商品、订单等表结构
3. **图书馆管理系统** - 实现图书借阅业务逻辑
4. **博客数据库** - 设计文章、用户、评论、标签等表
5. **数据分析查询** - 编写复杂的统计分析查询

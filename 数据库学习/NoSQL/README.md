# NoSQL 数据库学习

## 📖 学习内容

### NoSQL 概述
- NoSQL 数据库特点
- NoSQL vs SQL
- CAP 定理
- BASE 理论

### MongoDB

#### 基础概念
- 文档数据库
- 集合和文档
- BSON 数据格式

#### CRUD 操作
- 插入文档（insertOne, insertMany）
- 查询文档（find, findOne）
- 更新文档（updateOne, updateMany）
- 删除文档（deleteOne, deleteMany）
- 聚合管道（Aggregation Pipeline）

#### 高级特性
- 索引优化
- 数据建模
- 分片（Sharding）
- 复制集（Replica Set）
- 事务支持

### Redis

#### 数据类型
- String（字符串）
- Hash（哈希）
- List（列表）
- Set（集合）
- Sorted Set（有序集合）

#### 常用场景
- 缓存系统
- 会话存储
- 消息队列
- 排行榜
- 计数器
- 分布式锁

#### 高级功能
- 持久化（RDB, AOF）
- 发布订阅（Pub/Sub）
- 过期策略
- 内存优化

### 其他 NoSQL 数据库
- Cassandra（列式数据库）
- Neo4j（图数据库）
- Elasticsearch（搜索引擎）

## 🎯 学习目标

- [ ] 理解 NoSQL 数据库的特点和应用场景
- [ ] 掌握 MongoDB 的 CRUD 操作
- [ ] 能够设计 MongoDB 数据模型
- [ ] 熟练使用 Redis 进行缓存
- [ ] 掌握 Redis 常用数据类型和命令
- [ ] 了解不同 NoSQL 数据库的适用场景

## 💻 实践项目

1. **博客系统（MongoDB）** - 存储文章、用户、评论数据
2. **电商商品目录** - 使用 MongoDB 存储商品信息
3. **缓存系统** - 使用 Redis 实现数据缓存
4. **实时排行榜** - 使用 Redis Sorted Set 实现
5. **会话管理** - 使用 Redis 存储用户会话

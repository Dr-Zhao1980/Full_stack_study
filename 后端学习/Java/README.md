# Java 学习

## 📖 学习内容

### Java 基础
- Java 语法基础
- 面向对象编程（类、接口、继承、多态）
- 集合框架（List, Set, Map）
- 异常处理
- 多线程编程
- I/O 流操作
- Lambda 表达式和 Stream API

### Spring 框架生态

#### Spring Core
- 依赖注入（DI）
- 面向切面编程（AOP）
- Spring Bean 生命周期

#### Spring Boot
- 自动配置
- 起步依赖
- Spring Boot 应用创建
- 配置文件管理
- RESTful API 开发

#### Spring MVC
- 控制器和请求映射
- 视图解析
- 数据验证
- 拦截器和过滤器

#### Spring Data JPA
- JPA 实体映射
- Repository 接口
- 自定义查询
- 分页和排序

### 数据库操作
- JDBC 基础
- MyBatis ORM 框架
- Hibernate ORM 框架
- Spring Data JPA

### 进阶主题
- Spring Security（安全认证）
- Spring Cloud（微服务）
- 消息队列（RabbitMQ, Kafka）
- 缓存（Redis, Caffeine）
- 单元测试（JUnit, Mockito）

## 🎯 学习目标

- [ ] 掌握 Java 核心语法和面向对象编程
- [ ] 熟练使用 Spring Boot 开发 Web 应用
- [ ] 能够设计和实现 RESTful API
- [ ] 掌握 Spring Data JPA 进行数据持久化
- [ ] 理解 Spring Security 安全机制
- [ ] 了解微服务架构

## 💻 实践项目

1. **学生管理系统** - 实现学生信息的 CRUD 操作
2. **博客系统** - 使用 Spring Boot 开发完整博客应用
3. **电商平台** - 实现商品、订单、用户管理
4. **在线论坛** - 实现帖子发布、回复、点赞功能
5. **微服务项目** - 使用 Spring Cloud 构建微服务架构

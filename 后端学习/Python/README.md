# Python 学习

## 📖 学习内容

### Python 基础
- 变量和数据类型
- 条件语句和循环
- 函数和模块
- 列表、元组、字典、集合
- 文件操作
- 异常处理
- 面向对象编程

### Web 开发框架

#### Flask（轻量级）
- 路由和视图函数
- 模板引擎（Jinja2）
- 请求和响应对象
- 表单处理
- 数据库集成（SQLAlchemy）
- RESTful API 开发

#### Django（全功能）
- MTV 架构模式
- ORM 模型
- 视图和模板
- 表单和验证
- 用户认证系统
- Django Admin 后台
- 中间件

### 数据库操作
- SQLAlchemy ORM
- Django ORM
- 原生 SQL 查询
- 数据库迁移

### 进阶主题
- 异步编程（asyncio, FastAPI）
- RESTful API 设计
- Web 安全（CSRF, XSS, SQL 注入防护）
- 单元测试（pytest）
- 部署（Gunicorn, uWSGI）

## 🎯 学习目标

- [ ] 掌握 Python 基本语法和数据结构
- [ ] 熟练使用 Flask 或 Django 框架
- [ ] 能够设计和实现 RESTful API
- [ ] 掌握 ORM 进行数据库操作
- [ ] 理解 Web 安全基础
- [ ] 能够部署 Python Web 应用

## 💻 实践项目

1. **待办事项 API** - 使用 Flask 实现简单的 CRUD API
2. **博客系统** - 使用 Django 开发完整的博客应用
3. **图书管理系统** - 实现图书的增删改查和借阅功能
4. **微博克隆** - 实现用户动态、关注、点赞功能
5. **在线投票系统** - 实现投票创建、投票、结果展示

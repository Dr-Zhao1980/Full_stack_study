# Node.js 学习

## 📖 学习内容

### Node.js 基础
- Node.js 运行环境和安装
- 模块系统（CommonJS, ES Modules）
- npm 包管理器
- 文件系统操作（fs 模块）
- 路径处理（path 模块）
- 事件驱动（EventEmitter）
- 流（Streams）和缓冲区（Buffer）

### Web 服务开发
- HTTP 模块创建服务器
- Express.js 框架
  - 路由处理
  - 中间件机制
  - 请求和响应处理
  - 静态文件服务
- RESTful API 设计
- 身份认证（JWT, Session）
- 文件上传处理

### 数据库集成
- MongoDB（Mongoose ODM）
- MySQL（Sequelize ORM）
- Redis 缓存

### 进阶主题
- 异步编程和 Promise
- 错误处理和调试
- 性能优化
- 单元测试（Jest, Mocha）
- 部署和运维

## 🎯 学习目标

- [ ] 掌握 Node.js 核心模块使用
- [ ] 熟练使用 Express.js 构建 Web 应用
- [ ] 能够设计和实现 RESTful API
- [ ] 掌握身份认证和授权机制
- [ ] 理解异步编程模式
- [ ] 能够集成数据库进行数据持久化

## 💻 实践项目

1. **文件管理 CLI** - 使用 Node.js 创建命令行工具
2. **博客 API** - 实现文章的增删改查接口
3. **用户认证系统** - 实现注册、登录、JWT 认证
4. **实时聊天应用** - 使用 Socket.io 实现实时通信
5. **电商 API** - 完整的电商后端 API 服务

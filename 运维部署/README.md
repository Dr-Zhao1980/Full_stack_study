# 运维部署学习

## 📖 学习内容

### Git 版本控制

#### Git 基础
- Git 安装和配置
- 仓库初始化（init, clone）
- 基本操作（add, commit, push, pull）
- 分支管理（branch, checkout, merge）
- 标签管理（tag）
- 查看历史（log, diff）

#### GitHub / GitLab
- 远程仓库管理
- Pull Request / Merge Request 流程
- 代码审查
- Issues 管理
- GitHub Actions / GitLab CI

### Linux 基础
- 常用命令（ls, cd, mkdir, rm, cp, mv）
- 文件权限管理
- 进程管理
- 系统服务管理
- Shell 脚本编写

### Docker 容器化

#### Docker 基础
- Docker 安装
- 镜像和容器概念
- Dockerfile 编写
- 常用命令（build, run, ps, exec, logs）
- 数据卷（Volume）
- 网络配置

#### Docker Compose
- docker-compose.yml 编写
- 多容器编排
- 服务依赖管理

### Web 服务器
- Nginx 配置和使用
- 反向代理和负载均衡
- SSL/HTTPS 配置
- Apache 服务器

### 云服务部署

#### 云平台
- 阿里云
- 腾讯云
- AWS
- Azure
- Vercel / Netlify（前端部署）
- Heroku（应用部署）

#### 持续集成/持续部署（CI/CD）
- GitHub Actions
- GitLab CI/CD
- Jenkins
- Travis CI

### 监控和日志
- 应用日志管理
- 错误监控（Sentry）
- 性能监控
- 服务器监控

### 域名和 DNS
- 域名注册
- DNS 配置
- 域名解析

## 🎯 学习目标

- [ ] 掌握 Git 版本控制和团队协作流程
- [ ] 熟悉 Linux 基本操作
- [ ] 能够使用 Docker 容器化应用
- [ ] 掌握 Nginx 配置和使用
- [ ] 能够将应用部署到云服务器
- [ ] 了解 CI/CD 流程
- [ ] 掌握基本的监控和日志管理

## 💻 实践项目

1. **个人网站部署** - 使用 Nginx 部署静态网站
2. **Docker 化应用** - 将 Web 应用打包成 Docker 镜像
3. **全栈应用部署** - 部署前后端分离项目到云服务器
4. **CI/CD 流程** - 配置自动化测试和部署
5. **HTTPS 配置** - 为网站配置 SSL 证书
